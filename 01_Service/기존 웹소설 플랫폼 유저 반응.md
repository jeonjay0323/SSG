# 기존 웹소설 플랫폼 유저 반응

> 조사 일자: 2026-05-07  
> 출처: 인스티즈, 디시인사이드, 더쿠, 앱스토어 리뷰, 나무위키 등

---

## 😤 부정 반응

### 탐색 / 추천
- 프로모션이 랭킹을 오염 → "진짜 인기작이 뭔지 모르겠다"
- 추천이 취향과 안 맞고 광고성 추천과 실제 추천이 구분되지 않음

### 뷰어
- 스크롤 모드에서 페이지 경계 전환이 거슬림
- 폰트·간격 등 기본 커스터마이징 없는 플랫폼 존재 (카카오페이지)
- TTS 문장 사이 침묵이 과도하게 길어 어색함

### 광고
- 가상캐스팅(배우 연기) 광고에 대한 압도적 거부감
  - "원작 캐릭터랑 이미지가 안 맞음"
  - "몰입이 깨진다"
  - "코스프레 같고 거부감 듦"
  - "2D·3D는 아예 다른 세상인 느낌"
- 웹소설 읽는 도중 광고 노출 → 즉각적 불만

### 서재 / 결제
- 연재작 새 회차 구매 시 서재 분류가 초기화됨
- 앱 내 결제 불가 → 웹으로 이동 후 결제해야 함 (리디)
- 이용권 시스템 폐지 → 독자 반발 (네이버 시리즈)

### 기타
- PC 버전 사실상 방치 (카카오페이지)
- 앱 무거움, 크래시, 무한 로딩 등 안정성 문제
- 표지를 크게 볼 수 없음 (리디 앱)

---

## 👍 긍정 반응

- **기다리면 무료** — 유입 효과 압도적, 가장 성공한 과금 모델로 꼽힘
- **별점 / 댓글 시스템** — 작품 선택 기준으로 신뢰도 높음 (8.5 이상이면 볼 만하다는 암묵적 기준 형성)
- **이모티콘 댓글** — 카카오톡 연동으로 감정 표현이 다채로워 호평
- **TTS 기능 자체** — 이동 중·집안일 중 소비 패턴에 부합, 수요 높음
- **연관 작품 추천** — 작품 상세 하단의 유사작 추천 배치 호평
- **다크모드 뷰어** — 야간 독서가 많은 웹소설 특성상 필수 기능으로 인식

## 슥(SSG) 설계 시 시사점

| 피할 것 | 넣을 것 |
|---------|---------|
| 가상캐스팅·배우 광고 | 표지·삽화 기반 취향 추천 |
| 프로모션이 섞인 랭킹 | 투명한 인기 지표 분리 |
| 앱 내 결제 차단 | 원스텝 인앱 결제 |
| 서재 분류 초기화 | 연재작 상태 자동 유지 |
| 어색한 TTS 침묵 | 자연스러운 TTS 리듬 |
| 표지 확대 불가 | 표지 풀스크린 감상 지원 |
