# Landing Feature Spec

## 📑 Overview
서비스 첫 진입 화면. 풀스크린 캐러셀로 '슥(SSG)'의 정체성을 전달하고 카테고리별 Discovery 페이지로 유입을 유도합니다.

---

## 📐 Layout
* **화면 규격**: 풀스크린 (패드 기준 가로형 최적화)
* **UI 제한**: 헤더(Header) 및 푸터(Footer) 없음
* **공간 구성**: 캐러셀(Carousel)이 화면 전체를 점유

---

## 🧩 Components

### 1. Carousel
| 항목 | 내용 |
| :--- | :--- |
| **스타일** | Apple Music 스타일의 **3D 카드뷰** |
| **표시 카드 수** | 항상 3장 (좌 1 · 중앙 1 · 우 1) |
| **순환 방식** | 무한 루프 (Infinite Loop) |
| **Active 카드** | 중앙에 위치, 가장 크게 표시 |
| **Side 카드** | 좌우 각 1장, 축소 및 기울어짐 효과 적용 |

**[카드 구성 요소]**
* 카테고리별 지정 배경 영상
* 카테고리명
* 키워드 3개

**[인터랙션]**
* **전환**: 드래그 또는 스와이프로 카드 이동
* **선택**: Side 카드 클릭 시 해당 카드를 Active 상태로 전환
* **진입**: Active 카드 클릭 시 해당 카테고리의 `Discovery` 페이지로 이동

---

### 2. Background
* Active 카드 전환 시 전체 배경 그라디언트 동적 변경
* **전환 효과**: 1.2s Smooth Transition (부드러운 전환)

| 카테고리 | 배경 무드 (Gradient) |
| :--- | :--- |
| **추천** | 퍼플 / 신비로운 |
| **로맨스** | 핑크 / 로즈 / 따뜻한 |
| **로맨스 판타지** | 골드 / 퍼플 / 화려한 |
| **무협** | 먹색 / 레드 / 긴장감 |
| **판타지** | 다크블루 / 웅장한 |

---

### 3. Background Audio
* Active 카드 전환 시 카테고리 테마 배경음으로 자동 변경
* **재생 조건**: 브라우저 정책에 따라 첫 인터랙션 이후 재생 시작
* **효과**: 카드 전환 시 사운드 Fade Out → Fade In 적용

| 카테고리 | 배경음 (BGM) |
| :--- | :--- |
| **추천** | 잔잔한 로비 스타일 BGM |
| **로맨스** | 서정적인 피아노 선율 |
| **로맨스 판타지** | 화려하고 풍성한 오케스트라 |
| **무협** | 동양적인 악기 (중국풍) |
| **판타지** | 낮고 웅장한 오케스트라 |

---

### 4. Sound Toggle
* **위치**: 화면 우측 상단 고정
* **상태 표시**: 
    * 재생 중: 사운드 바(Sound Bar) 애니메이션 활성화
    * 일시정지: 애니메이션 정지 및 아이콘 흐리게(Opacity) 처리
* **기능**: 탭(Tap) 시 사운드 온/오프 토글

---

### 5. Dot Indicator
* **위치**: 하단 중앙 고정
* **기능**: 
    * Active 카드에 해당하는 Dot 하이라이트 표시
    * Dot 탭 시 해당 순서의 카드로 즉시 이동

---

## ⚠️ Edge Cases
| 상황 | 처리 방안 |
| :--- | :--- |
| **브라우저 자동재생 차단** | 사용자 첫 클릭/스와이프 인터랙션 발생 후 재생 시작 |
| **배경음 없는 카테고리** | 현재 재생 중인 음악을 Fade Out 처리 후 정지 |
| **영상 로딩 지연** | 해당 카테고리의 컬러 그라디언트를 Fallback 배경으로 노출 |
