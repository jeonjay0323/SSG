## 독자 사이드 화면 목록

| 화면명 | 역할 | 비고 |
| :--- | :--- | :--- |
| **Landing** | 서비스 정체성 표현 및 카테고리 진입 | 3D 캐러셀 UX |
| **Preferences** | 맞춤형 삽화 스타일 · TTS 스타일 선택 | 온보딩 필수 단계 |
| **Discovery** | 인기작, 신작, 랭킹 및 키워드 기반 탐색 | 필터링 기능 |
| **Novel Detail** | 작품 상세 정보, 삽화/영상 미리보기, 회차 목록 | |
| **Viewer** | 몰입형 독서 환경 (사운드, 삽화, TTS 컨트롤) | 서비스 핵심 경험 |
| **Event** | 프로모션 및 혜택 안내 | 기획 예정 |
