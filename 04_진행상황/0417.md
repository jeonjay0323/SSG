## 🔍 화면별 상세 기획

### 1. Landing (대문)
* **Visual**: Apple Music 스타일의 **풀스크린 3D 캐러셀**.
* **Interaction**: 카드 전환 시 장르별 **배경 영상과 배경음(BGM)** 자동 변경.
* **Action**: 카드 클릭 시 해당 카테고리가 필터링된 `Discovery` 페이지로 이동.
* **Card 구성**:

| 카테고리 | 키워드 | 배경음 (BGM) |
| :--- | :--- | :--- |
| **추천** | #지금핫한 #신작 #완결 | 잔잔한 로비 BGM |
| **로맨스** | #설레임 #재벌 #순애 | 서정적인 피아노 선율 |
| **로맨스 판타지** | #황녀 #빙의 #집착남주 | 화려한 오케스트라 |
| **무협** | #고수 #복수 #강호 | 동양적인 악기 (중국풍) |
| **판타지** | #마법 #던전 #회귀 | 웅장한 오케스트라 |

### 2. Preferences (취향 설정)
* **Step 1**: 생성형 AI 기반 **삽화 스타일** 선택 (실사, 애니메이션 등).
* **Step 2**: 선호하는 **TTS 보이스 스타일** 선택.
* 최초 가입 시 1회 진행하며, 설정에서 상시 변경 가능.

### 3. Viewer (몰입형 뷰어)
* **Context-Aware**: 소설 내 분위기 태그를 분석하여 **배경 이미지와 사운드** 실시간 변환.
* **Control Panel**:
    * `TTS`: 온/오프 및 속도 조절
    * `BGM`: 배경음 토글
    * `Illustration`: AI 삽화 표시 여부 선택
